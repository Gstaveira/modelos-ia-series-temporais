# modelos-ia-series-temporais

Repositório contendo os códigos de implementação e os datasets utilizados na previsão de séries temporais com modelos de Inteligência Artificial.

## Modelos incluídos
- **ARMA**
- **LSTM**
- **Random Forest**
- **XGBoost**
- **MLP**

# Como rodar os códigos
Os códigos podem ser executados no ambiente do Google Colab. Deve-se fazer o upload dos datasets na plataforma e inserir o caminho correto
do dataset na variável arquivo = 'caminho do dataset'.
